# 代理列表页 UI 设计

## 1. 页面定位

代理列表页用于集中展示和管理 TCP、UDP、HTTP、HTTPS 代理资源。

## 2. 路由

- ` /proxies `

## 3. 页面目标

- 浏览代理资源及运行状态
- 支持创建、筛选、排序与分页
- 提供启停等快速生命周期操作
- 支持跳转代理详情页

## 4. 页面结构

- 页面标题区
- 工具栏
- 代理表格区
- 分页区

## 5. 工具栏设计

包含：

- 搜索框
- 用户筛选
- 客户端筛选
- 类型筛选
- 状态筛选
- 排序控件
- 手动刷新按钮
- “创建代理”按钮

## 6. 列表字段

- 代理 ID
- 名称
- 类型
- 所属用户
- 所属客户端
- 状态
- 运行状态
- EntryHost
- EntryPort
- TargetHost
- TargetPort
- 活跃连接数
- 上传流量
- 下载流量
- TCP 错误数
- UDP 错误数
- HTTP 错误数
- 创建时间
- 更新时间

## 7. 布局设计

- 桌面端使用宽表格布局
- 状态、类型与运行状态使用标签样式区分
- 移动端转为卡片模式
- 创建按钮固定为主要操作

## 8. 创建代理交互

创建入口使用弹窗或抽屉，按代理类型动态显示字段：

- 通用字段：名称、所属用户、所属客户端、描述
- TCP/UDP：EntryHost、EntryPort、TargetHost、TargetPort
- HTTP/HTTPS：Host、TargetHost、TargetPort

## 9. 列表交互

- 点击行进入 ` /proxies/:id `
- 列表行可提供启用/禁用快捷操作
- 页面保持 5 秒轮询刷新运行态和统计数据
- 创建或启用 TCP/UDP 代理冲突时，明确展示 `ENTRY_CONFLICT`

## 10. 状态设计

### 10.1 初始加载
- 表格骨架屏

### 10.2 空数据态
- 暂无代理

### 10.3 无结果态
- 当前筛选条件无匹配代理

### 10.4 校验失败
- 表单字段级错误提示

### 10.5 冲突态
- 对端口冲突单独展示，说明监听已被占用

### 10.6 列表错误态
- 显示“代理列表加载失败”

## 11. 可用性要求

- 类型切换时表单字段变化应平滑明确
- 表格宽字段支持截断和悬浮提示
- 快捷启停操作需避免误触，可使用确认机制

## 12. 验收标准

- 支持四类代理资源浏览
- 支持创建与快速启停
- 支持搜索、筛选、排序、分页
- 对 `ENTRY_CONFLICT` 有清晰 UI 提示
