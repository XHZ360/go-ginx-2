# 变更记录

变更过程用单文件记录，不再拆成 OpenSpec 的 proposal/design/tasks/specs。

## 目录约定

```text
changes/
  active/      进行中的变更（按需创建）
  completed/   已完成、仍可能参考的变更
  archive/     历史批次与过期续作笔记
```
